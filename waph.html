Course Information:The "Web Application Programming and Hacking" course covers languages like HTML5, JavaScript, CSS, PHP, and MySQL and offers a thorough overview of front-end and back-end web development. It emphasizes secure programming principles and uses practical exercises to examine vulnerabilities and attacks in web applications. Web and web application development, client-side and server-side programming, Git version control, and DevOps procedures are among the subjects covered in the syllabus. Web application security is also covered in detail in the course. A project-based approach that includes labs, hackathons, individual projects, and a team project focused on developing a real-world application deployed in the cloud. Lab 1 | Lab 2 | Hackathon 1

Lab 1 Overview: This lab focuses on understanding HTTP requests and responses using Wireshark, Telnet, and CGI programs. Learn More

Lab 2 Overview: This lab covers basic front-end development with HTML, CSS, and JavaScript, including creating web pages, digital and analog clocks, and making AJAX requests, working with APIs. Learn More

Hackathon 1 Overview: This hackathon focuses on identifying cross-site scripting (XSS) attacks and adopting OWASP guidelines to prevent XSS events. Learn More
